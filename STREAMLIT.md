# 🌐 Streamlit Application

This project includes a **Streamlit-based web interface** for the Book Recommendation System.  
The application allows users to interact with the recommendation model through a simple and interactive UI.

---

# 📊 Features

- Interactive user interface
- Book selection through dropdown
- Real-time book recommendations
- Simple and user-friendly design

---

# 🧠 How It Works

The Streamlit app connects to the recommendation logic implemented in the project.  
It performs the following steps:

1. Loads the processed dataset.
2. Accepts a selected book from the user.
3. Computes similar books using the recommendation model.
4. Displays the top recommended books in the interface.

---

# 🛠 Technologies Used

- Streamlit
- Pandas
- NumPy
- Scikit-learn
